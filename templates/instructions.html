Instructions to Use the Excel Compare Tool

  1. Select your Source, Target, and Property files from your device using the file input fields provided.
  2. Ensure that your source and target files are in either .xlsx or .xls format, and your property file is in .properties format.
  3. The Property file should contain properties such as:
    • sourceColumns: Names of columns in the source file, separated by commas. For example: sourceColumns=Name, Address, Age
    • targetColumns: Names of corresponding columns in the target file, separated by commas. For example: targetColumns=Name, Address, Age
    Create the Property file using a text editor and save it with a .properties extension.
  4. Before comparing the Excel files using the tool, ensure to format the Excel cells by removing unnecessary cells, data, or any unnecessary formatting.
  5. Click the "Compare Files" button to start the comparison process.
  6. As soon as the comparison is completed, a text file (.txt) format will be downloaded to your device with the filename result_file, from which you can analyze the results.
